Population CalculatorTutorial

This following page will take you through step by step on how to use the API that we used effectively and how to use the visualisation for this page as well.

For this page, there is one Ajax request to get the data with as soon as the page is loaded.

An Ajax request main purpose is to update a webpage asynchronously, meaning a user will not have to refresh a webpage for a tiny change to take effect. There are four main features to an ajax request the first would be to update a webpage without reloading it, the second would be to request data from a server after the page has loaded up, the third would be to receive data after a page has loaded up and the forth would be to send data to a server and this would happen in the background.

To use an Ajax request we first need to implement jQuery. jQuery is a lightweight JavaScript library, this means that writing in JavaScript will become a lot easier as with this library implemented tasks that use to require a lot of lines of code to function now require less as the jQuery library will most likely have useful functions that the developer could use. The jQuery library will also help simplify the Ajax requests that will be seen throughout this website.

The API call that is used to gain the data for this website is the following:
//restcountries.eu/rest/v2/all
Compared to our other APIs that are used this one doesn’t require any additional variables as well as not needing an API key.

The information we are looking for in this array of data is each country’s population size and the countries corresponding alpha2 code. The alpha2 code of a country is its unique two letter code. We gain the population size and alpha2 code by using a for loop to go through each position in the array of data we gained through the API and store each countries alpha2 code and population in the variables that we have made.



The visualisation that we are using for this page is a map of the world and the data is being shown through the colour of each country as the darker red a country the more population it has. We are doing this by first importing the map from google and then loading the map on the screen. After this, we use the google function drawRehionMap() to populate the map with the information we previously gained and then we declare what the colours are going to be and this is done within the same function. For the colours, we need to declare the min value which is 10000 and the max value which is 5000000 and then we declare what each sections colour will be. For this case, we have declared that the min value will be a salmon colour and the max value will be a dark red. The colour section when complete will then go through each section and colour it in with a corresponding colour using the colour range that we have declared earlier.
