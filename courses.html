Path Finder

To use this page, you simply must choose a qualification by using the drop-down box below. Once you have chosen a qualification a donut chart will automatically be shown below the drop-down menu. The doughnut chart simply shows the percentage of people in the specified occupation with the qualification that has been chosen.
User can click on a section in the chart to see the sections job in a job finder website (Mobile is Double Click)
User can cycle through the dropdown box by using the keyboards left and right arrow keys.

For a more in-depth tutorial on how to fully take advantage of this page please follow the link here to our pathfinder tutorial page.
